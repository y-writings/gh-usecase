# Nix Flake Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a minimal Nix flake that builds and runs the `gh-usecase` Go CLI.

**Architecture:** Mirror the `y-writings/xapi-usecase` flake shape with one package and one app per supported system. Use `pkgs.buildGoModule` for `./cmd/gh-usecase`, but set a fixed `vendorHash` because this repository has external Go dependencies in `go.sum`.

**Tech Stack:** Nix flakes, nixpkgs `buildGoModule`, Go 1.26.3 module build.

---

## File Structure

- Create `flake.nix`: flake definition, package output, app output, Go module build metadata.
- Create `flake.lock`: generated by Nix when evaluating the flake input.

### Task 1: Add Minimal Flake

**Files:**
- Create: `flake.nix`

- [x] **Step 1: Create the initial flake with a fake vendor hash**

Create `flake.nix` with this content:

```nix
{
  description = "gh-usecase CLI";

  inputs = {
    nixpkgs.url = "github:NixOS/nixpkgs/nixos-unstable";
  };

  outputs = { self, nixpkgs }:
    let
      systems = [
        "x86_64-linux"
        "aarch64-linux"
        "x86_64-darwin"
        "aarch64-darwin"
      ];

      forEachSystem = f:
        nixpkgs.lib.genAttrs systems (system:
          f system (import nixpkgs { inherit system; })
        );
    in
    {
      packages = forEachSystem (_system: pkgs:
        let
          gh-usecase = pkgs.buildGoModule {
            pname = "gh-usecase";
            version = "0.0.0";

            src = pkgs.lib.cleanSourceWith {
              src = ./.;
              filter = path: _type:
                let
                  rel = pkgs.lib.removePrefix "${toString ./.}/" (toString path);
                in
                rel == "go.mod"
                || rel == "go.sum"
                || rel == "cmd"
                || rel == "internal"
                || pkgs.lib.hasPrefix "cmd/" rel
                || pkgs.lib.hasPrefix "internal/" rel;
            };
            subPackages = [ "cmd/gh-usecase" ];

            vendorHash = pkgs.lib.fakeHash;

            ldflags = [
              "-s"
              "-w"
            ];

            meta = {
              description = "Small Go CLI for GitHub repository use cases";
              homepage = "https://github.com/y-writings/gh-usecase";
              license = pkgs.lib.licenses.mit;
              mainProgram = "gh-usecase";
            };
          };
        in
        {
          inherit gh-usecase;
          default = gh-usecase;
        });

      apps = forEachSystem (system: _pkgs:
        let
          gh-usecase = {
            type = "app";
            program = "${self.packages.${system}.gh-usecase}/bin/gh-usecase";
          };
        in
        {
          inherit gh-usecase;
          default = gh-usecase;
        });
    };
}
```

- [x] **Step 2: Run the build to discover the dependency hash**

Run: `nix build .`

Expected: FAIL with a fixed-output hash mismatch that prints a `got: sha256-...` value for `vendorHash`.

- [x] **Step 3: Replace the fake hash with the printed hash**

Edit `flake.nix` so this line:


becomes:

```nix

```

Use the exact `sha256-+0MZl/+ybYYCiOdQXpEji6IDXuLAnbo1G1djX1sT3Uc=` value from the `got:` line, including quotes in the Nix file.
